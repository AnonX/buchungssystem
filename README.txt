Starten der Anwendung:

2. Ggf. Flask / abhängige Packages und Libraries lokal importieren
3. In main.py die Run Configuration (run.app()) starten
4. Flask-interner WSGI Entwicklungsserver startet, Webanwendung kann über localhost http://127.0.0.1:5000
aufgerufen werden
